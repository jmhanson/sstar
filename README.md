# sstar
Siemens (spoke) star target vector image generator.

## Description
This is a standalone Python script that generates a PostScript file `sstar.ps` with a Siemens star target vector image. The star target can have an arbitrary number of spokes.

![Siemens star target example](sstar-example.png)

See also https://en.wikipedia.org/wiki/Siemens_star

## Useage
`sstar.py [n]`

*n*     Desired integer number of spokes. Minimum: 2. Default: 20.

## Dependencies
Python 3.2 or greater.
