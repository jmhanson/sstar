# sstar
Siemens (spoke) star target vector image generator.

## Description
This is a standalone python script that generates a PostScript file with a Siemens star target vector image with an arbirtrary number of spokes. 

## Useage
`sstar [n]`

    *n* Desired number of spokes. Minimum: 2. Default: 20.

## Dependencies
Python 3.2 or greater.
